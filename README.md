Python tool to scan subdomains of a target site
It checks if the subdomains present in the .txt wordlist exist on the target site, returning the subdomain IP

> Require python 3.9x

## How to Use

```bash
python main.py [domain] -w [wordlist]
```

## Simbols

- ```[+] - Request found```
- ```[-] - Request Error```
- ```[=] - Result```
